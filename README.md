# House Price Analysis

This project analyzes the House Price dataset to determine key factors affecting property values.

## Structure

```
📁 House_Price_Analysis
 ├── 📂 data (house_prices.csv)
 ├── 📂 Src (notebook - Assignment_3_Part_3.ipynb)
 ├── 📂 results (executive_summary.pdf)
 ├── README.md
```

## Key Findings

- **House Size**: Larger homes (GrLivArea, TotalBsmtSF) correlate with higher prices.
- **Quality & Age**: Newer homes and those with higher Overall Quality (OverallQual) are valued more.
- **Garage Area**: Larger garages contribute to higher sale prices.
- **Price Distribution**: House prices are right-skewed, indicating a small number of expensive properties.
